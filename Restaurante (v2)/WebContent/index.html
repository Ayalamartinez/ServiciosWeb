Listado de Servicios Web - Restaurante

Servicios de Clientes


      Crear cliente (POST- Restaurante/rest/createCliente)
      Obtener un cliente (GET - estaurante/rest/getCliente/{emailCliente})
      Obtener todos los clientes (GET - Restaurante/rest/getClientesAll)
      Actualizar un cliente (POST - Restaurante/rest/updateCliente/{emailCliente})
      Eliminar un cliente (DELETE - Restaurante/rest/deleteCliente/{emailCliente})
      Login de cliente (GET - Restaurante/rest/login)
      Cambiar password de cliente (POST - Restaurante/rest/cambiarPassword)

Servicios de Ingredientes


      Crear ingrediente (POST - Restaurante/rest/ingrediente/createIngrediente)
      Obtener un ingrediente (GET - Restaurante/rest/ingrediente/getIngrediente/{idIngrediente})
      Obtener todos los ingredientes (GET - Restaurante/rest/ingrediente/getIngredientesAll)
      Actualizar un ingrediente (POST - Restaurante/rest/ingrediente/updateIngrediente/{idIngrediente})
      Eliminar un ingrediente (DELETE - Restaurante/rest/ingrediente/deleteIngrediente/{idIngrediente})
      Descontar del stock de un ingrediente (POST - Restaurante/rest/ingrediente/descontarIngrediente/{idIngrediente}/{cant})

Servicios de Menús


      Crear menú (POST - Restaurante/rest/menu/createMenu)
      Obtener un menú (GET - Restaurante/rest/menu/getMenu/{idMenu})
      Obtener todos los menúes (GET - Restaurante/rest/menu/getMenusAll)
      Agregar un plato en un menú (POST - Restaurante/rest/menu/addPlatoMenu/{idMenu}/{idPlato})
      Eliminar un plato en un menú (POST - Restaurante/rest/menu/deletePlatoMenu/{idMenu}/{idPlato})
      Eliminar un menú (DELETE - Restaurante/rest/menu/deleteMenu/{idMenu})

Servicios de Pedidos


      Crear pedido (POST - Restaurante/rest/pedido/createPedido)
      Obtener un pedido (GET - Restaurante/rest/pedido/getPedido/{idPedido})
      Obtener todos los pedidos (GET - Restaurante/rest/pedido/getPedidosAll)
      Agregar detalle de un pedido (POST - Restaurante/rest/pedido/addDetallePedido/{idPedido})
      Eliminar detalle de un pedido (DELETE - Restaurante/rest/pedido/deleteDetallePedido/{idPedido}/{idDetalle})
      Eliminar un pedido (DELETE - Restaurante/rest/pedido/deletePedido/{idPedido})
      Actualizar el estatus de un pedido (POST - Restaurante/rest/pedido/updateEstatusPedido/{idPedido}/{estatus})

Servicios de Platos


      Crear plato (POST - Restaurante/rest/plato/createPlato)
      Obtener un plato (GET - Restaurante/rest/plato/getPlato/{idPlato})
      Obtener todos los platos (GET - Restaurante/rest/plato/getPlatosAll)
      Actualizar un plato (POST - Restaurante/rest/plato/updatePlato/{idPlato})
      Agregar un ingrediente a un plato (POST - Restaurante/rest/plato/addIngredientePlato/{idIngrediente}/{idPlato})
      Eliminar un ingrediente a un plato (DELETE - Restaurante/rest/plato/deleteIngredientePlato/{idPlato}/{idIngrediente})
      Eliminar un plato (DELETE - Restaurante/rest/plato/deletePlato/{idPlato})

Servicios de Restaurantes

      Obtener un restaurante (GET - Restaurante/rest/restaurante/getRestaurante/{idRestaurante})
